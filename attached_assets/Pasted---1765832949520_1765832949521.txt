╠════════════════════════════════════════════════════════════════════════╣
║ 1. Purpose                                                             ║
║ - Interactive Forex/XAU/USD dashboard                                   ║
║ - Live placeholder signals (replace with bot later)                     ║
║ - Automatic WIN/LOSS logging                                           ║
║ - Trading journal with CSV export                                       ║
║ - Telegram bot integration (token/chat ID saved locally)                ║
║ - Fully modular for future upgrades                                     ║
╠════════════════════════════════════════════════════════════════════════╣
║ 2. Tech Stack                                                          ║
║ - Frontend: React.js + Tailwind CSS                                     ║
║ - Animations: Framer Motion                                             ║
║ - Charts: Recharts / sparklines                                         ║
║ - State: React Context / Zustand                                        ║
║ - Local Storage: bot token, chat ID, default pairs/timeframes, journal  ║
║ - Backend (optional for persistence): Node.js + Express                ║
║ - Placeholder API for signals (replace later with bot data)             ║
╠════════════════════════════════════════════════════════════════════════╣
║ 3. Folder Structure / Files                                            ║
║ - /src                                                                 ║
║   ├─ index.js (ReactDOM render, load App)                               ║
║   ├─ App.js (Main dashboard layout + panels)                            ║
║   ├─ components/                                                        ║
║   │   ├─ SignalCard.js (Live signals placeholder)                       ║
║   │   ├─ JournalTable.js (Sortable/filterable journal)                  ║

║   │   ├─ TopBar.js (Timeframe selector, market selector, settings btn)  ║
║   │   └─ Alerts.js (Floating notifications)                              ║
║   └─ utils/                                                             ║
║       ├─ localStorage.js (save/load settings & journal)                 ║

║ 4. Features / Functionality                                             ║
║ - Signal Panel: cards showing pair, direction, entry, SL/TP, confidence ║
║ - Signal click → opens detailed modal with mini chart                   ║
║ - Journal Panel: auto-log signals, sortable, filterable, export CSV     ║
║ - Settings Panel: save/load bot token, chat ID, default pairs/timeframes║
║ - Alerts: floating notifications for new signals & expiry               ║
║ - Countdown timers for each signal card                                  ║
║ - Local storage ensures settings & journal persist on reload             ║
║ - Dark/Light mode toggle                                                 ║
║ - Fully responsive, immersive, clickable, animated UI                    ║
║ - Modular: new signals, pairs, charts, AI features can be added later   ║
╠════════════════════════════════════════════════════════════════════════╣
║ 5. Replit Setup                                                        ║
║ - Create a new Replit → Import from GitHub (use your gilgalo-trading-   ║
║   dashboard repo)                                                        ║
║ - Install Node.js (Replit default)                                       ║
║ - Run: npm install / npm start                                           ║
║ - Dashboard will open → live placeholder signals displayed               ║
║ - Local storage will remember bot token, chat ID, journal automatically ║
║ - You can click settings to update bot token/chat ID without touching AI║
